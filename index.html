Instalasi Komputer

Kemampuan memasang, mengatur, dan mengkonfigurasi perangkat keras dan sistem operasi komputer agar siap digunakan.

HTML (HyperText Markup Language)

Skill membuat struktur dasar halaman web, seperti teks, gambar, link, dan tabel menggunakan tag HTML.

CSS (Cascading Style Sheets)

Kemampuan mendesain tampilan web (warna, layout, font, dsb) agar lebih menarik dan responsif.

JavaScript (JS)

Kemampuan menambahkan interaktivitas pada web, seperti validasi form, slider, pop-up, dan animasi.

Jaringan Komputer (Network)

Skill dalam merancang, membangun, dan mengkonfigurasi jaringan LAN/WAN, termasuk IP addressing dan troubleshooting jaringan.

Desain Web

Kemampuan merancang tampilan dan user interface (UI) web yang menarik dan mudah digunakan dengan memperhatikan prinsip UX (User Experience).
